# 日本房价地图 Phase 2 UX Refinement SPEC

## 1. 目标

修正第二轮预览里用户直接指出的 4 个前台问题：

- 默认只给山手线核心大站保留站名，其余站点降为图标级展示
- 房产均价模式下，价格直接显示在地图图标上
- 右下图例不长期占视线，改成可自动缩小
- 站点卡片支持点击空白区域自动收起

## 2. 范围

- 调整站点种子数据的默认标签层级
- 调整价格模式 marker 文案
- 调整图例交互
- 调整站点卡片交互

## 3. 交付标准

- 价格模式下，用户不用点开卡片就能直接看到大致价格
- 默认地图不再被大面积说明卡和长期展开图例挡住
- 点中站点后卡片会弹出
- 点击地图空白处后卡片会收起
- 构建、测试、lint 全通过

## 4. 非目标

- 不接正式官方全量站点
- 不改商业化和 AI 路线
- 不进下一轮数据导入脚本
