# 日本房价地图 Phase 2 SPEC

## 1. 目标

在第一轮单页地图骨架的基础上，推进第二轮最小可交付：

- 把当前前端写死的种子数据外置到 `public/data/`，改成运行时加载
- 保留现有单页地图和模式切换交互，不重写页面
- 提供一个稳定的 Tailnet 内网预览入口，确保 Windows 设备可直接访问

## 2. 本轮为什么要做

第一轮已经证明：

- 单页地图工具形态成立
- 车站锚点成立
- 图层切换成立

第二轮要证明的是：

- 页面层已经和数据层解耦
- 后续正式导入官方数据时，不需要再改前端主结构
- 项目可以通过 Tailnet 给本地 Windows 直接预览

## 3. 范围

- 新增 `public/data/tokyo/` 下的 JSON 数据目录
- 把站点、学校、风险区数据改成从 JSON 加载
- 增加加载态和错误态
- 增加 `preview:tailnet` 运行脚本
- 增加启动/停止 Tailnet 预览脚本
- 实际启动预览并生成可访问 URL

## 4. 交付标准

- `npm test` 通过
- `npm run lint` 通过
- `npm run build` 通过
- 页面仍能正常切换模式和选择站点
- 预览服务能在当前机器上后台运行
- 能给出一条从 `home-windows.tail4b5213.ts.net` 可访问的 Tailnet 网址

## 5. 非目标

- 不在这一轮接官方全量数据
- 不做后端数据库
- 不做站点详情页
- 不做 AI
