# 日本房价地图 V1.8 Kanto Frame SPEC

> 当前状态真相源最终仍以项目根 `README.md` 为准。本 Spec 只服务于本轮连续升级。

## 1. 目标

把地图视角从“东京核心局部”切到“关东底图范围”，但数据范围仍保持东京，不扩到其他城市。

## 2. 已确认问题

- 当前默认视角仍是东京核心局部，和用户期望的“先给关东范围”不一致。
- 当前可拖拽边界没有收口到关东，用户可以继续把地图拉到与当前项目范围无关的更大区域。
- 当前前台 copy 仍把重置动作说成“东京视角”，和新的地图边界不一致。

## 3. 本轮范围

### 必做

- 默认地图视角切到关东范围。
- 拖拽边界限制在关东范围。
- 重置动作切到关东视角。
- README 与前台文案同步当前真相：
  - 底图范围先给关东
  - 数据仍只做东京

### 本轮不做

- 不增加东京以外城市站点数据。
- 不增加东京以外城市图层。
- 不重建新的 PMTiles 区域切片文件。

## 4. 验收

- 本地打开默认进入关东范围，而不是东京核心近景。
- 地图不能随意拖出关东范围。
- 搜索、模式切换、站点打开、空白关闭等现有行为不退化。
- `npm test`
- `npm run lint`
- `npm run build`
- `npm run acceptance:tokyo-v1`
- README 回写当前范围真相并保持仓库干净
