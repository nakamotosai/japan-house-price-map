# 日本房价地图 Phase 1 SPEC

## 1. 目标

从零启动一个新项目，落地第一轮可运行版本：

- 新建项目目录 `日本房价地图`
- 采用开源地图方案
- 直接进入东京地图
- 布局参考 Google Maps 的工具感
- 优先验证“车站锚点 + 图层切换 + 轻量面板”的底座

## 2. 用户要求

- 不做单独首页，打开就是地图
- 工具栏压在边缘，交互尽量简单
- 默认显示东京地图与车站
- 模式切换本质上是图层切换
- 后续新增数据时，不应该重写页面，只需要挂新图层

## 3. 第一轮范围

- 单页 React 应用
- MapLibre GL JS
- 日本官方地理院公开瓦片底图
- 左侧边栏
- 左上搜索栏
- 地图主视图
- 车站搜索
- 站点点击面板
- 模式切换系统
- 首批 5 个模式：
  - 房产均价
  - 公示地价
  - 车站热度
  - 学校
  - 灾害风险

## 4. 数据策略

- 第一版用前端种子数据跑通交互与图层底座
- 数据结构必须从一开始就贴近未来正式数据导入
- 站点数据必须单独建主索引
- 模式必须注册化，不能把逻辑硬编码在页面里

## 5. 交付标准

- `npm run build` 通过
- 页面可本地启动
- 地图能显示东京
- 模式按钮能切换不同图层
- 站点搜索能定位到已有站点
- 点击站点能看到相应摘要信息
- README 能说明当前状态和运行方式

## 6. 非目标

- 不接入正式后端
- 不接入实时官方 API
- 不做完整站点详情页
- 不做 AI 功能
- 不做大阪和全国
