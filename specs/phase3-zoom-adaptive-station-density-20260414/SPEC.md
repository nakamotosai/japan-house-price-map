# 日本房价地图 Phase 3.1 Zoom Adaptive Station Density SPEC

## 1. 目标

把车站 marker 的显示密度改成“随缩放级别逐层展开”，解决默认视图下一次性显示过多车站导致地图发灰、价格重点被淹没的问题。

## 2. 范围

- 基于地图缩放级别控制站点 marker 是否显示
- 基于站点等级、换乘能力、真实客流、价格覆盖情况决定优先显示顺序
- 让默认缩放只保留核心站和重点价格站
- 放大后逐层补出次级站和全部站点

## 3. 交付标准

- 默认缩放下不再一股脑显示 500+ 个站点圆点
- 放大地图后，站点数量会明显增加
- 核心大站和已有价格覆盖站在远景下仍保持可见
- 现有点击交互、价格 marker、站点卡片不回退
- 测试、lint、build、预览验证通过

## 4. 非目标

- 不在这一轮改学校、灾害、房价数据来源
- 不在这一轮改站点详情页
- 不在这一轮做聚类动画或服务端瓦片化
